# kolm-start

### Tanstack Start template with tRPC, Tailwind

- `@tanstack/start v1` https://tanstack.com/start/latest
- `@trpc v11` https://trpc.io/docs
- `tailwindcss v4` https://tailwindcss.com/docs/v4-beta

## Getting started

```bash
git clone
npm install
npm run dev
```
